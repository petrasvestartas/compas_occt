# compas_occt

Nanobind wrapper for OCCT

## Installation

Stable releases can be installed from PyPI.

```bash
pip install compas_occt
```

To install the latest version for development, do:

```bash
git clone https://github.com//compas_occt.git
cd compas_occt
pip install -e ".[dev]"
```
