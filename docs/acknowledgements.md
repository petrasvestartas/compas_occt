# Acknowledgements

`compas_occt` follows the public API of [`compas_occ`](https://github.com/compas-dev/compas_occ),
the OpenCASCADE-based NURBS and Brep backend for [COMPAS](https://compas.dev) developed by the
[Block Research Group (BRG)](https://block.arch.ethz.ch/) at ETH Zurich. The class names, method
signatures, docstrings, examples, and plugin behaviour are modelled directly on that work, and we
gratefully acknowledge the BRG and the COMPAS community for designing and maintaining it.

`compas_occt` is built on:

- [OpenCASCADE Technology (OCCT)](https://github.com/Open-Cascade-SAS/OCCT) — the geometry kernel.
- [nanobind](https://github.com/wjakob/nanobind) — the C++/Python binding layer.
- [COMPAS](https://compas.dev) — the computational framework and plugin system.
